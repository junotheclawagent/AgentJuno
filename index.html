Honest answers

Things you might be wondering.

How is this different from ChatGPT?

ChatGPT forgets every conversation. Juno keeps a memory file on your own server. ChatGPT chats. Juno acts (with permission) — reads emails, runs scheduled tasks, drafts replies. ChatGPT is rented; Juno is owned.

What if I'm not technical?

Then this is for you. We do the install. We hand you a manual written for the curious, not for engineers. You text the agent in Telegram. That's the entire user interface.

Where does my data live?

On a server in an EU country of your choice (default Helsinki). GDPR-protected. We don't host it; we don't see it after handover. You own the keys.

Can it be hacked?

Anything can be hacked. We hardened the obvious surfaces: gateway loopback-only, filesystem sandboxed at the OS level, no skill installs without manual review, daily spend caps that physically refuse runaway calls. The Risk Acknowledgment lists every residual risk we know about.

What if I want to cancel?

Delete the server from your Hetzner dashboard. Sixty seconds. Everything is gone. We don't hold anything. There's nothing to cancel because there's no subscription unless you're on the retainer — and that's monthly, cancel anytime.
